# 校园题库
> 欢迎大家一起构建属于自己学校的专属题库🎉

## 使用须知
1. 校园题库中存在的题目将被题库配置优先调用，且不会被外校调用。
2. 用户需要绑定学校，并完成`学生认证`。
3. 题库有封禁机制，请不要上传与所在学校无关的题目。

## 题库机制
### 1. 归档
   - 只有文件夹可以被归档
   - 归档后，其他用户无法编辑，只能查看
   - 文件夹创建者可以直接归档或取消归档，非创建者归档或取消归档需要发起投票
### 2. 垃圾站
   - 题目删除后会先进入垃圾站，三天后永久删除，在此之前，皆可以恢复
### 3. 投票
   - 文件夹的归档，取消归档，删除操作需要发起投票
   - 课程的删除需要发起投票
   - 投票人数超过一半，投票通过
### 4. 封禁
   - 封禁后的用户将无法使用校园题库，包括查看答案与编辑
   - 其他使用不受限制

## 如何快速上传题目

我写了一个油猴脚本来支持快速上传题目到校园题库  
[https://scriptcat.org/zh-CN/script-show-page/5270](https://scriptcat.org/zh-CN/script-show-page/5270)  

他会出现在页面的右下角，在选择课程与试卷后，点击解析，然后点击上传，脚本就会自动将页面中的题目上传至校园题库对应的文件夹。下次再次使用OCS网课助手时，就会优先调用校园题库中的题目，且不会消耗次数。
<div class="pic">
  <img src="/images/题库上传脚本.png" alt="题库上传脚本.png" />
</div>

2. 中国大学Mooc（作业，检测）
