# 校园题库
> 用户绑定学校后可以访问。绑定后不可以解绑。每位用户只能查看属于自己学校的题库。欢迎大家一起构建属于自己学校的专属题库🎉

## 使用须知
1. 用户需要绑定学校，并完成`学生认证`。
2. 校园题库中存在的题目将被题库配置优先调用，且不会被外校调用。
3. 题库有封禁机制，请不要上传与所在学校无关的题目。

## 题库机制
### 1. 归档
   - 只有文件夹可以被归档
   - 归档后，其他用户无法编辑，只能查看
   - 文件夹创建者可以直接归档或取消归档，非创建者归档或取消归档需要发起投票
### 2. 垃圾站
   - 题目删除后会先进入垃圾站，三天后永久删除，在此之前，皆可以恢复
### 3. 投票
   - 文件夹的归档，取消归档，删除操作需要发起投票
   - 课程的删除需要发起投票
   - 投票人数超过一半，投票通过
### 4. 封禁
   - 封禁后的用户将无法使用校园题库，包括查看答案与编辑
   - 其他使用不受限制
