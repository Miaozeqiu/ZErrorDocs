</div>

ZError是一款已获OCS网课助手授权，专为网课设计的题库，包括**在线**与**本地**题库。在线题库提供多层次的api调用服务，包含`校园题库-公共题库-第三方题库-AI`,为网课提供便捷的题库解决方案。本地题库通过软件，支持所有格式的AI调用，提供便捷的题库管理功能，包括增删改查，分类管理等。

::: tip 提示
如果你是网课新手，建议直接选择[在线题库](https://tiku.zerror.cc)，无需安装任何软件。网课插件使用[OCS网课助手](https://docs.ocsjs.com/) https://docs.ocsjs.com/
:::
## 核心功能

### 🔍 AI题库
- 用预设的提示词返回适配OCS的答案
- 将历史查询题目缓存到本地题库
- 自义定端口对局域网开放

### 📖 本地题库管理
- 对本地题库的增删改查
- 利用文件夹系统对题目进行分类
